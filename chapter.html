Chapter Summary

  • All Html elements can have attributes
  • The href attribute of < a > specifies the URL of the page the link goes to
  • The src attribute of <img > specifies the path to the image to be displayed
  • Thewidth and height attributes of<img > provide size information for images
  • Thealt attribute of <img > provides an alternate text for an image
  • Thestyle attribute of the<Html > tag declares the language of the Web page
  • Thelang attribute of the <Html > tag declares the language of the Web page
  • Thetitle attribute defines some extra information about an element
